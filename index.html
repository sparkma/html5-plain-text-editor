HTML5 plain text editor

demo page

API function calls are binded to the buttons below:

Click any paragraph or heading to edit.

During the click plaint-text-editor attaches to the <P> or <H...>.

After you could simply click any button to test an API.

The bindings and sample of using are held in file js/scenario.js

Actual code is located in js/plain.text.editor.js

I'm trying to keep code fairly simple. Any comments or wishes are welcome.

Plain text editor uses only contenteditable tag. No designmode applied.

By the way decorating editor as jQuery plug-in or in any other way is a simple thing.

implemented API functions

getText/setText - works after being initialized via click on a paragraph or heading.

getSelection - works after setting a selection over text.

undo/redo - works only for editing - this is a limitation from DOM Range.
